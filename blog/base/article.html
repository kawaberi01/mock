読みやすさを高めるUIルール集

デザイン 2025-08-10 / 更新: 2025-08-12

目次

この記事では、UIの読みやすさを高めるための基本原則を紹介します。余白、タイポグラフィ、色のコントラスト、情報の階層化など、実践的なテクニックを幅広く取り上げます。

1. 余白設計の基本

余白は要素間の関係性を示し、情報の塊を識別しやすくします。コンポーネント内外で一貫したスケールを用いましょう。

  • 8px グリッドなどの等比スケールを使う
  • コンテンツ密度に応じて余白を調整する

実例: カードレイアウト

カードの内側余白とカード同士の外側余白は、情報のグループ化に大きく寄与します。

2. タイポグラフィ

本文は 16px 前後、line-height: 1.6～1.8 を基準に、読みやすさを優先します。

見出しの階層

見出しはサイズだけでなく、余白・太さ・色で階層を表現します。

3. コントラストと配色

テキストと背景のコントラスト比は WCAG の推奨値以上を保ちます。アクセントカラーは必要十分に。

可読性の鍵は「過不足のない強調」。目立たせたい情報だけを強調すること。

4. スクロールアニメーション

Intersection Observer を活用して、パフォーマンスに優れたアニメーションを実現します。以下はシンプルな例です。

const io = new IntersectionObserver((entries) => {
  entries.forEach((e) => {
    if (e.isIntersecting) {
      e.target.classList.add('show');
      io.unobserve(e.target);
    }
  });
});
document.querySelectorAll('.reveal').forEach(el => io.observe(el));

5. まとめ

読みやすさは、ルールと一貫性で作られます。最小の装飾で最大の効果を狙いましょう。
